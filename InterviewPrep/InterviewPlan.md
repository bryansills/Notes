##### Interview Tips

Write code in classes  
Write test cases  
Don't dive right in.  
Communicate.  
Speech:

* Talk slow
* Eye contact
* Talk about positive things
* Ask for feedback

##### UCTWV (Vidhur's Method)
Understand, Clarify, Test Cases, Write it, Validate


##### Problem Solving

* Always have a plan
* Restate the problem
* Divide the problem
* Start with what you know
* Reduce the problem
* Looking for analogies
* Experiment
* Don't get frustrated
